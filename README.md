Title:
Phishing Email Detection & Awareness System (Task 2)

Overview:
This project analyzes a phishing email sample to identify common phishing indicators and provides awareness guidance for users. The goal is to help organizations reduce phishing risks through education and clear reporting.

Scope:
-Educational analysis only
-No live systems tested
-No exploitation or interaction with malicious links

Tools Used:
-Google Message Header Analyzer / MXToolbox (simulated header analysis)
-Browser inspection (link and domain review)

Methodology:
-Analyzed a phishing email sample
-Reviewed sender domain, message content, and embedded link
-Simulated email header analysis
-Identified phishing indicators
-Classified risk and documented prevention guidelines

Deliverables:
-Phishing Detection & Awareness Report
-Supporting evidence screenshots
